# Color Match Game 🎨

A mini JavaScript game where players recreate a randomly generated RGB color using sliders.

## Features
- Random target color
- Live RGB slider adjustment
- Real-time accuracy percentage
- Responsive UI

## Built With
- HTML
- CSS
- JavaScript

## How it Works
The player adjusts Red, Green, and Blue sliders to match the target color. Accuracy is calculated by comparing RGB values.
